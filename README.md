pubsub-message-wall
===================

A message wall powered by XMPP pubsub (demo)

## What does it do?

* Logs in
* Sets presence
* Subscribes to a node
* Listens for posts to a node
* Drops data into UI

Note: Only handles the most basic of pubsub items for demo

## Instructions

* Copy config.example.js to config.js
* Edit to suit
* `npm i .`
* grunt connect
* http://localhost:4000
